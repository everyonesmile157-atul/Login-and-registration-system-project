# Login and Registration System

## Description
This is a Login and Registration System developed in C++ using File Handling.

## Features
- User Registration
- User Login
- Username Validation
- Password Validation
- Data stored in CSV file
- Input Validation

## Technologies Used
- C++
- File Handling
- OOP
- VS Code

## How to Run
1. Open the project in VS Code,
2. Compile the project,
3. Run main.cpp
